# Conference Website Starter

This is a static conference website template for GitHub Pages.

## Files

- `index.html`: home page
- `program.html`: program page
- `speakers.html`: invited speakers page
- `registration.html`: registration information
- `abstract.html`: abstract submission information
- `venue.html`: venue and travel information
- `organization.html`: committees and sponsors
- `contact.html`: contact page
- `css/style.css`: design and responsive layout
- `js/main.js`: mobile navigation and automatic year
- `files/abstract-template.txt`: simple abstract template

## Replace these placeholders

- `ICoN10`
- dates and venue
- `conference@example.com`
- `https://forms.gle/REPLACE_WITH_YOUR_REGISTRATION_FORM`
- `https://forms.gle/REPLACE_WITH_YOUR_ABSTRACT_FORM`
- speaker names, institutions, and photos

## GitHub Pages publishing

Use Settings -> Pages -> Deploy from a branch -> main -> /(root).
